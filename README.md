## 📖 Overview

Smart Finance Hub is a modern expense management platform designed to simplify how people manage shared finances.

Whether you're planning a trip, sharing an apartment, organizing an event, or managing household expenses, Smart Finance Hub allows users to create groups, record expenses, split bills using multiple methods, track balances in real time, and settle debts transparently.

The platform supports multi-currency transactions, interactive financial analytics, and a scalable architecture built with React 19, TypeScript, and NestJS.

The project emphasizes clean architecture, reusable components, type safety, and modern full-stack engineering practices.
