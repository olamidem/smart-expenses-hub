## 📖 Overview

Smart Expenses Hub is a modern full-stack expense management platform designed to simplify how individuals, friends, families, roommates, and teams manage shared finances.

Whether you're planning a trip, sharing household bills, organizing an event, or managing everyday expenses, Smart Expenses Hub makes it easy to record expenses, split bills fairly, track balances in real time, and settle debts transparently.

The platform supports multiple expense splitting methods, multi-currency transactions, interactive financial analytics, and a scalable architecture built with React 19, TypeScript, and NestJS.

This project showcases modern software engineering practices, including clean architecture, reusable components, type safety, responsive design, and scalable backend development.

## ✨ Features

### 👥 Group Management

- Create and manage expense groups
- Invite members or add guest participants
- Support multiple groups simultaneously

### 💸 Smart Expense Splitting

- Equal split
- Exact amount split
- Percentage split
- Share-based split
- Automatic balance calculation

### 🌍 Multi-Currency Support

- Live exchange rates
- Automatic currency conversion
- Group default currency
- Currency history

### 📊 Analytics Dashboard

- Spending by category
- Monthly trends
- Interactive charts
- Financial insights

### 🤝 Settlement Management

- Track who owes whom
- Suggested settlements
- Record payments
- Live balance updates

### 🔐 Authentication

- Secure JWT Authentication
- Protected routes
- Guest Mode
- User profiles

### 📱 Modern User Experience

- Responsive design
- Dark mode
- Smooth animations
- Form validation
